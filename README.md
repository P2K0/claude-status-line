# ⚡️ Claude Status Line

> **Claude Status Line** 是一款专为 **Claude Code** 用户打造的极简、现代化的可视化配置中心。 / A minimalist and modern visual configuration hub for **Claude Code** users.

## ✨ 核心特性 / Features

- **🚀 🚀 Zero-Threshold Configuration**: Break away from tedious Shell scripts and customize your status bar via an intuitive Web UI. / 摆脱繁琐的 Shell 脚本编写，通过直观的 Web 界面自定义你的状态栏。
- **👁️ 所见即所得 / WYSIWYG**: Built-in terminal emulator for real-time preview of color schemes, icons, and Git layouts. / 内置精准的终端模拟器，实时预览配色、图标和 Git 状态。
- **🎨 专业级美学 / Professional Aesthetics**: Curated themes like Amber, Emerald, Cyberpunk, and Rosepine. / 提供多款精心调配的主题色。
- **🛠️ 深度集成 / Deep Integration**: One-click installation scripts, auto-handling `settings.json` for macOS/Linux. / 一键安装脚本，自动处理环境配置。
- **📊 实时状态 / Real-time Metrics**: Visual progress bars, token statistics, and Git indicators. / 支持进度条可视化、Token 统计、Git 状态显示。

## 📖 使用指南 / Guide

1. Adjust **styles**, **colors**, and **Git modes** in the right panel. / 在右侧面板调整样式、配色和 Git 模式。
2. Preview results in the left terminal emulator. / 左侧查看实时预览。
3. Click **"Copy Install Script"** and run it in your terminal. / 点击 "Copy Install Script" 并在终端运行。
