## ✨ 核心特性

- **🚀 零门槛配置**：摆脱繁琐的 Shell 脚本编写，通过直观的 Web 界面自定义你的状态栏。
- **👁️ 所见即所得**：内置精准的终端模拟器，实时预览配色、图标和 Git 状态布局。
- **🎨 专业级美学**：提供 Amber, Emerald, Cyberpunk, Rosepine 等多款精心调配的主题色。
- **🛠️ 深度集成**：一键生成 Base64 加密安装脚本，自动处理针对 macOS/Linux 的环境配置与 `settings.json` 注入。
- **📊 实时状态**：支持进度条可视化、Token 消耗统计、多级 Git 变更显示。

## 📖 使用指南

1. 在右侧面板调整 **进度条风格**、**主题配色**、**Git 显示模式** 等选项。
2. 左侧终端模拟器将实时反馈你的调整结果。
3. 点击 **"Copy Install Script"**，在终端中粘贴运行，即可完成 Claude Code 的集成。
