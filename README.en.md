## ✨ Core Features

- **🚀 Effortless Configuration**: Customize your status line through an intuitive Web UI without writing complex Shell scripts.
- **👁️ WYSIWYG Editor**: Built-in accurate terminal emulator to preview color schemes, icons, and Git status layouts in real-time.
- **🎨 Professional Aesthetics**: Features curated themes like Amber, Emerald, Cyberpunk, Rosepine, and more.
- **🛠️ Deep Integration**: Generate Base64-encoded installation scripts with one click, automatically handling environment setup and `settings.json` injection for macOS/Linux.
- **📊 Real-time Metrics**: Visual progress bars, token consumption statistics, and multi-level Git change indicators.

## 📖 Get Started

1. Adjust **Progress Bar Style**, **Color Preset**, **Git Mode**, and other settings in the right panel.
2. Preview the changes instantly in the terminal emulator on the left.
3. Click **"Copy Install Script"**, paste it into your terminal, and run it to integrate with Claude Code.
